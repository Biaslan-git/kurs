=================================================================================
1. УСТАНОВКА PYTHON
=================================================================================

1. Скачай Python 3.8 или новее с официального сайта: https://www.python.org/downloads/
2. При установке ОБЯЗАТЕЛЬНО поставь галочку "Add Python to PATH"
3. Проверь установку: открой командную строку (Win+R → cmd) и введи:
   
   python --version
   
   Должна появиться версия Python (например: Python 3.11.0)

=================================================================================
2. УСТАНОВКА ЗАВИСИМОСТЕЙ
=================================================================================

ВАРИАНТ 1: Через uv (РЕКОМЕНДУЕТСЯ - быстрее и современнее)
------------------------------------------------------------
1. Установи uv (если ещё нет):
   
   Windows (PowerShell):
   powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
   
   Linux/Mac:
   curl -LsSf https://astral.sh/uv/install.sh | sh

2. Открой командную строку в папке certificate_system

3. Установи зависимости:
   
   uv sync
   
   Всё готово! uv автоматически создаст виртуальное окружение.

ВАРИАНТ 2: Через pip (классический способ)
-------------------------------------------
1. Открой командную строку в папке с проектом:
   - Зажми Shift + правая кнопка мыши на папке certificate_system
   - Выбери "Открыть окно PowerShell здесь" или "Открыть в Терминале"

2. Установи зависимости командой:
   
   pip install -r requirements.txt
   
   Установится:
   - PyQt6 (графический интерфейс)
   - python-docx (для создания Word документов)

=================================================================================
4. РАБОТА С ПРИЛОЖЕНИЕМ
=================================================================================

ПЕРВЫЙ ЗАПУСК:
--------------
1. При первом запуске автоматически создастся база данных в папке database/
2. Откроется главное окно с кнопками быстрого доступа

ПОСЛЕДОВАТЕЛЬНОСТЬ ДЕЙСТВИЙ:
-----------------------------

ШАГ 1: Добавь адреса
   - Нажми "Справочник адресов"
   - Заполни поля: Город, Улица, Дом, Корпус (необязательно), Квартира
   - Нажми "Добавить"
   - Можно добавить несколько адресов

ШАГ 2: Добавь жильцов
   - Нажми "Справочник жильцов"
   - Заполни все поля: ФИО, дату рождения, паспортные данные
   - Нажми "Добавить"
   - Добавь всех нужных жильцов

ШАГ 3: Зарегистрируй жильцов по адресам
   - Нажми "Регистрация жильцов"
   - Выбери адрес из выпадающего списка
   - Выбери жильца
   - Укажи дату регистрации и степень родства
   - Нажми "Зарегистрировать"
   - Повтори для всех жильцов

ШАГ 4: Сформируй справку
   - Нажми "Сформировать справку"
   - Выбери адрес
   - Укажи дату выдачи, кто выдал и получателя
   - Нажми "Сформировать предпросмотр" - увидишь как выглядит справка
   - Нажми "Сохранить в Word" - справка сохранится в .docx файл

ГОРЯЧИЕ КЛАВИШИ:
----------------
- F1          - Справка (в некоторых окнах)
- Enter       - Подтверждение
- Esc         - Закрыть окно
- Tab         - Переход между полями
- Ctrl+F      - Поиск (где доступен)

=================================================================================
5. СОЗДАНИЕ .EXE ФАЙЛА (для сдачи курсовой)
=================================================================================

Чтобы создать исполняемый файл, который можно запускать без установки Python:

1. Установи PyInstaller:
   
   pip install pyinstaller

2. Перейди в папку проекта и выполни:
   
   pyinstaller --onefile --windowed --name="Certificate_System" src/main.py

3. Готовый .exe появится в папке dist/

ВАЖНО: 
- При запуске .exe база данных создастся в той же папке где лежит .exe
- Скопируй database/ рядом с .exe если нужно перенести базу

7. ВОЗМОЖНЫЕ ПРОБЛЕМЫ И РЕШЕНИЯ
=================================================================================

ПРОБЛЕМА: "python не является внутренней или внешней командой"
РЕШЕНИЕ: Python не добавлен в PATH. Переустанови Python с галочкой "Add to PATH"

ПРОБЛЕМА: "ModuleNotFoundError: No module named 'docx'"
РЕШЕНИЕ: Установи зависимости: pip install -r requirements.txt

ПРОБЛЕМА: Окна открываются некорректно / не видно элементов
РЕШЕНИЕ: Проверь масштабирование экрана в Windows (должно быть 100%)

ПРОБЛЕМА: При создании .exe файл слишком большой
РЕШЕНИЕ: Это нормально, PyInstaller упаковывает весь Python. Можно сжать архиватором.

ПРОБЛЕМА: База данных не сохраняется
РЕШЕНИЕ: Проверь права доступа к папке. Запусти от администратора.
